# diozaz-gadgets — Elite Tech Storefront

A sophisticated gadget marketplace featuring elite tech components, built with Next.js for high performance and a premium shopping experience.

## 🚀 Key Features
- **Elite Tech Catalog:** Curated selection of high-end gadgets and components.
- **Dynamic Routing:** Next.js App Router for fast, seamless navigation.
- **Modern UI:** Clean, tech-focused aesthetic with responsive layouts.
- **Search & Filter:** Advanced product discovery and categorization.

## 🛠️ Tech
- Next.js
- React
- Tailwind CSS
- Lucide React
